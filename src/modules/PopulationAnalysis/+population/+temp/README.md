# One-off functions to solve EXTREMELY specific problems

This folder/namespace contains some one-off functions that were created to solve extremely specific problems that came up during pilots and testing. Be extremely cautious about using these functions and consider whether a more general solution would be more appropriate for your use case.

Again, unless you have the exact same problem as the one solved by these functions, it is strongly recommended to avoid using them in your own analyses.

Please refer to the individual function documentation for more details on what each function does and the specific problem it addresses.
